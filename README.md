# ☕ Çiçek Kafe

Çiçek Kafe, HTML ve CSS kullanılarak geliştirilmiş modern ve responsive bir kafe sitesi projesidir.

## 📌 Projenin Amacı
Bu proje, kullanıcıların kafenin menüsünü kolayca inceleyebileceği, sade ve şık bir tasarıma sahip web sitesi oluşturmak amacıyla geliştirilmiştir.

## 📂 Site Bölümleri
Sitemiz 3 ana bölümden oluşmaktadır:

- 🏠 Anasayfa
- 📋 Menü
- ℹ️ Hakkımda

## ☕ Menü İçeriği
Kafemizde bulunan ürün kategorileri:

- ☕ Sıcak İçecekler
- 🥤 Soğuk İçecekler
- 🍰 Tatlılar
- 🥐 Unlu Mamuller
- 🍪 Atıştırmalıklar

## ✨ Özellikler
- Responsive (mobil uyumlu) tasarım
- Modern ve kullanıcı dostu arayüz
- Sade ve şık tasarım
- Kolay gezinme deneyimi

## 🛠️ Kullanılan Diller
- HTML5
- CSS3

## ⚡ Notlar
- Projede yalnızca HTML ve CSS kullanılmıştır.
- Bazı bölümlerin geliştirilmesinde yapay zekadan destek alınmıştır.
- Proje eğitim ve portföy amacıyla hazırlanmıştır.
